probabilistic (bayesian) statistical modeling in forest biometrics.

home / teaching / own materials

a model describes data that one could observe from a system. if we use probability to express all sorts of uncertainty associated with our model, then inverse probability (Bayes rule) allows us to learn from measurements: infer unknown quantities, adapt our models, make predictions, ... .
... and a probabilistic programming language like https://mc-stan.org/ is a tool that allows us to practically get there.
