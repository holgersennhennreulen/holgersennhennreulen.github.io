Notizen zur Normalverteilung.

Holger Sennhenn-Reulen [uncertaintree.github.io]

16. Februar 2021

Wenn eine Zufallsvariable sowohl positive als auch negative stetige Werte annehmen kann, so ist die Normalverteilung häufig das erste Modell das uns in den Sinn kommt um den unbekannten, datengenerierenden Prozess zu beschreiben.

H\u00e4ufig wird die Normalverteilung auch auf Gr\u00f6\u00dfen angewandt die ausschlie\u00dflich positiv - seltener auch ausschlie\u00dflich negativ - sind, Die Normalverteilung kann auch eine pragmatische Wahl sein wenn sich in der Anwendung Quantile in den ung\u00fcltigen Wertebereich hinein verlagern; der Erhalt der G\u00f6ltigkeit dieses Modells h\u00e4ngt dann mit der St\u00e4rke der Verletzung des Vorwissens ab und muss in jedem Anwendungsfall erneut eingeordnet werden. Oft kann aber eine andere Verteilungsform, wie etwa die Gamma-Verteilung, hier die bessere Modellierungsleistung liefern.

Die Normalverteilung hat zwei Parameter, \(\mu\) fuer den Erwartungswert - das ist der Lageparameter -, und \(\sigma\) fuer die Standardabweichung - das ist der Skalierungsparameter. Die umgekehrte Standardabweichung, \(1/\sigma\) ist auch als Praezision bekannt.

Wenn eine Zufallsvariable \(Y\) als der Normalverteilung folgend angenommen wird, so schreiben wir: $$ Y\sim\text{Normal}(\mu,\sigma^2), $$ und sagen: Die Zufallsvariable Y folgt einer Normalverteilung mit Erwartungswert mu und Varianz sigma Quadrat.

R-code: 

mu <- 3
sigma <- 1.27
x_seq <- seq(-5, 5, by = .01)
par(las = 1, bty = "n", mgp = c(2.5, .7, 0), mar = c(3.5, 3.5, 1.5, 0) + .1)
main_title <- expression(paste("Normal", 
                               group("(",list(mu==3, sigma^2==1.27^2),")")))
plot(x = x_seq, y = dnorm(x = x_seq, mean = mu, sd = sigma), type = "l", 
     xlab = "x", ylab = "Dichte von X an der Stelle x", main = main_title)
y_lab <- expression(paste("W'keit f\u00fcr ", X <= x))
plot(x = x_seq, y = pnorm(q = x_seq, mean = mu, sd = sigma), type = "l", 
     xlab = "x", ylab = y_lab, main = main_title, ylim = c(0, 1))
